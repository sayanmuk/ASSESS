ASSESS ~ Analysis of Sample Set Enrichment Scores

Overview

Analysis of Sample Set Enrichment Scores (ASSESS) is a computational method that determines whether an a priori defined set of genes shows statistically significant, concordant differences between two biological states (e.g. phenotypes).

Documentation: Information on the ASSESS method including publications.

Software: Implementations of the method plus additional resources to analyze, annotate and interpret enrichment results.

Genesets: View and download genesets.
